# Screenshots 说明

此目录包含各个示例的截图。

## 如何添加 sample6 的截图

1. 运行 sample6 示例：
   - 在浏览器中打开 `packages/tsl/examples/src/sample6/index.html`
   - 或者通过开发服务器访问该示例

2. 截图步骤：
   - 等待示例完全加载并显示旋转的纹理立方体
   - 使用截图工具（如 Windows 的 Snipping Tool 或浏览器开发者工具）截取画布区域
   - 保存为 `sample6.png` 并放置在此目录

3. 截图建议：
   - 建议尺寸：与其他示例截图保持一致（参考 sample2.png 和 sample4.png）
   - 格式：PNG
   - 内容：应显示旋转的纹理立方体，包含 WebGPU 和 WebGL 两个画布
