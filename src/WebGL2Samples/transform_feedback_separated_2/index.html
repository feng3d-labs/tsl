WebGL2 Samples - transform_feedback_separated_2

此示例使用 Transform Feedback 实现简单的粒子系统。粒子从中心发射，受重力加速度影响下落，
当粒子超出边界或生命周期结束时会重新生成。

关于示例

  • 使用 TSL (TypeScript Shader Language) 编写着色器
  • 同时支持 WebGL 和 WebGPU 渲染
  • 使用 Transform Feedback 分离模式（SEPARATE_ATTRIBS）
  • WebGPU 使用 Compute Shader 模拟 Transform Feedback
  • 1000 个粒子实时更新和渲染
  • 粒子位置、速度、生成时间、生命周期分别存储在不同缓冲区
  • 使用 ping-pong 缓冲区交换策略
